This interaction reflects the tension between anticipation and completion in digital conversation.

typing… waiting… sending… repeating. 

The bubbles move in a continuous loop to express how messaging is ongoing and unending in everyday digital life. deliberately used 'unsent' as my Boolean variable to express the 
emotional state of a message before it arrives. Inside the loop, a conditional checks the value of unsent to determine the color of all bubbles, visually representing whether the messages were still forming,. or they'd been delivered
